# ATS-ROCKET

Simple implementation inspired by the ROCKET algorithm (for **R**and**O**m **C**onvolutional **KE**rnel **T**ransform) as described in the [original article published by Angus Dempster, Francois Petitjean, and Georey I. Webb.](https://arxiv.org/abs/1910.13051)

## Sources
1. ["ROCKET: Exceptionally fast and accurate time series classication using random convolutional kernels" by Angus Dempster, Francois Petitjean, and Georey I. Webb.](https://arxiv.org/abs/1910.13051)
2. [PyTorch - Training a Classifier](https://pytorch.org/tutorials/beginner/blitz/cifar10_tutorial.html)
3. [PyTorch - Conv1D Documentation](https://pytorch.org/docs/stable/generated/torch.nn.Conv1d.html)
4. ["A guide to convolution arithmetic for deep learning" by Vincent Dumoulin, and Francesco Visin](https://arxiv.org/pdf/1603.07285.pdf)
5. [Online Archive of Time Series Datasets for Classification](http://timeseriesclassification.com/dataset.php)
